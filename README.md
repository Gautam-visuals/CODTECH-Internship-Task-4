OVERVIEW OF PROJECT:

Project : SIMPLE TEXT EDITOR

Objective : The primary objective of this project is to build an interactive, console-based text processing utility that combines textual statistics gathering with a dynamic find-and-replace engine.

### Technical Learning Objectives:
* **Dynamic String Splicing:** Learn how to manipulate character strings in C without causing memory corruption, specifically when swapping text segments of varying lengths.
* **Substring Boundary Validation:** Implement logical constraints to distinguish between complete standalone words and partial matches inside larger words (e.g., finding "cat" without modifying "category").
* **State-Based Counting:** Master text analysis logic using flag variables to track transitions from whitespace to character zones for precise word and sentence extraction.
* **Buffer-to-Buffer Reconstruction:** Practice building output strings step-by-step using secondary scratchpad arrays (`buffer`), a fundamental concept in secure C data handling.
* **Robust Standard I/O Mixing:** Gain hands-on experience cleanly switching between multi-word line inputs (`fgets`) and single-word token gathering (`scanf`) by managing the standard input stream safely.
